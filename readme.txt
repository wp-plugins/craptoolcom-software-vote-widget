== Description ==
With this Widget you can show the actual Worst Software Rating from CrapTool.com in your sidebar.
== Installation ==

1. Upload `craptool_widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Design/Widget and add the Widget
4. Edit the Widget
